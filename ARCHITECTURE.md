# Memento-X 架构文档

## 一、产品定位

Memento 是一款面向专业影视创作者的 AI 视频编辑工具，核心逻辑是：

> "AI 只做意图理解（自然语言 → 结构化工作流），像素级工作全部由确定性工具完成。"

用户只做一件事：在本地界面说出需求，系统自动调度工具完成工作。

## 四、完整工作流

用户输入："把这个人换成钢铁侠，背景改成火星"

```
【本地】UI界面捕获指令 → 发送到云端
         │
【云端】AI意图理解引擎（通义千问VL-Pro）
        解析为 JSON 工作流
         │
【本地】调度器接收 JSON 工作流
         │
         ├── BiRefNet → 抠出人物
         ├── SAM2-MLX → 追踪遮罩（24帧）
         ├── ComfyUI → 生成钢铁侠+火星背景
         ├── FFmpeg → 合成帧序列
         ├── DaVinci → 调色（如已安装）
         └── HyperFrames → 添加字幕（如有）
         │
【本地】输出 4K ProRes / H.264 MP4 成片
```

AI 只参与第一步：理解用户意图 → 生成工作流。剩余所有步骤：确定性工具执行（0元调用费）。

## 六、设计原则

1. AI只做意图理解，不做像素生成
2. 所有像素级工作由确定性工具完成
3. 云端存储核心逻辑（闭源，不可破解）
4. 本地执行所有重计算（利用用户硬件）
5. 数据不出本地（视频素材留在用户电脑）
6. 订阅制商业模式（非一次性买断）
7. 所有工具按需下载（启动器管理）
8. 用户只做一件事：说出需求
