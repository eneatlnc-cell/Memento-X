# Memento-X

> AI 视频编辑工具 — AI 只做意图理解，像素级工作全部由本地确定性工具完成。

## 一句话定位

用户在本地界面说出需求 → 云端 AI 解析为结构化工作流 → 本地调度器调用工具库 → 输出 4K 成片。

## 核心理念

```
AI 只做一件事：理解用户意图（自然语言 → JSON 工作流）
所有像素级工作：确定性工具（BiRefNet / SAM2 / ComfyUI / FFmpeg）
单次处理成本：~0.01-0.05 元（仅 AI 意图理解）
视频素材：不出本地
```

## 快速开始

```bash
# 云端
cd cloud && pip install -r requirements.txt && python main.py

# 本地
cd local && npm install && npm run dev
```

## 成本结构

| 环节 | 费用 | 运行位置 |
|------|------|----------|
| 意图理解 (通义千问VL-Pro) | ~0.01-0.05元/次 | 云端 |
| SVG场景编辑 (BiRefNet) | 0元 | 本地 |
| 追踪 (SAM2-MLX) | 0元 | 本地 |
| 特效 (ComfyUI) | 0元 | 本地 |
| 合成 (FFmpeg) | 0元 | 本地 |
| 调色 (DaVinci) | 0元 | 本地 |
| 字幕 (HyperFrames) | 0元 | 本地 |
| **总计** | **~0.01-0.05元/次** | |

## 设计原则

1. AI 只做意图理解，不做像素生成
2. 所有像素级工作由确定性工具完成
3. 云端存储核心逻辑（闭源）
4. 本地执行所有重计算（利用用户硬件）
5. 数据不出本地（视频素材留在用户电脑）
6. 订阅制商业模式
